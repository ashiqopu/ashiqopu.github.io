2014-2019
Problem: Minimize wireless broadcast data transmissions and overall
Vehicle-to-RSU communication latency in VANETs to provide improved
road-safety and infotainment.
Primary solution: Network coded wireless broadcast with optimized
scheduling algorithms for time sensitive data request
Sub-solution-1: Co-operative RSU-to-RSU cache information transfer
which excludes explicit upload by vehicles.
Sub-solution-2: Coding for heterogeneous data with weighted moving
average of maximum data size in a clique.
Sub-solution-3: Time-sensitive heuristic coding for heterogeneous data
in real-time multi-item query.
Results: Significant lower latency and wireless broadcast overhead
with high data-retrieval rate.
